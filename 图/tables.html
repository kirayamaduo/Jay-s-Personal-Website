数据库表结构设计

users（用户表）
字段名 类型 约束 说明
id BIGINT PK 主键，自增
username VARCHAR(50) UKNN 用户名
password_hash VARCHAR(255) NN 密码哈希
email VARCHAR(100) UK 邮箱地址
avatar_url VARCHAR(255) 头像URL
role VARCHAR(20) 角色(ADMIN/GUEST)
last_login TIMESTAMP 最后登录时间
created_at TIMESTAMP 创建时间
updated_at TIMESTAMP 更新时间
posts（博客文章表）
字段名 类型 约束 说明
id BIGINT PK 主键，自增
author_id BIGINT FK 作者ID→users
title VARCHAR(255) NN 文章标题
slug VARCHAR(255) UKNN URL友好标识
excerpt VARCHAR(500) 文章摘要
content LONGTEXT Markdown内容
rendered_content LONGTEXT 渲染后HTML
cover_image VARCHAR(500) 封面图片
is_published BOOLEAN 是否发布
is_featured BOOLEAN 是否精选
view_count INT 浏览次数
published_at TIMESTAMP 发布时间
created_at TIMESTAMP 创建时间
comments（评论表）
字段名 类型 约束 说明
id BIGINT PK 主键，自增
post_id BIGINT FKNN 文章ID→posts
parent_id BIGINT FK 父评论ID(回复)
nickname VARCHAR(50) NN 评论者昵称
email VARCHAR(100) 评论者邮箱
website VARCHAR(255) 评论者网站
content TEXT NN 评论内容
reply_to_nickname VARCHAR(50) 回复目标昵称
is_approved BOOLEAN 是否审核通过
is_spam BOOLEAN 是否垃圾评论
ip_address VARCHAR(45) IP地址
created_at TIMESTAMP 创建时间
projects（项目作品表）
字段名 类型 约束 说明
id BIGINT PK 主键，自增
title VARCHAR(100) NN 项目标题
slug VARCHAR(100) UK URL友好标识
description VARCHAR(500) 项目简介
content LONGTEXT 详细说明
icon VARCHAR(50) 项目图标
gradient_class VARCHAR(100) 渐变色样式
github_url VARCHAR(255) GitHub链接
demo_url VARCHAR(255) 演示链接
stars_count INT Star数量
tech_stack JSON 技术栈数组
is_featured BOOLEAN 是否精选
display_order INT 显示顺序
tags（标签表）
字段名 类型 约束 说明
id BIGINT PK 主键，自增
name VARCHAR(50) UKNN 标签名称
slug VARCHAR(50) UK URL友好标识
color VARCHAR(20) 标签颜色
description VARCHAR(200) 标签描述
created_at TIMESTAMP 创建时间
post_tags（文章-标签关联表）
字段名 类型 约束 说明
post_id BIGINT PKFK 文章ID→posts
tag_id BIGINT PKFK 标签ID→tags
project_tags（项目-标签关联表）
字段名 类型 约束 说明
project_id BIGINT PKFK 项目ID→projects
tag_id BIGINT PKFK 标签ID→tags
messages（消息表）
字段名 类型 约束 说明
id BIGINT PK 主键，自增
name VARCHAR(100) NN 发送者姓名
email VARCHAR(100) NN 发送者邮箱
subject VARCHAR(200) 消息主题
content TEXT NN 消息内容
is_read BOOLEAN 是否已读
is_replied BOOLEAN 是否已回复
replied_at TIMESTAMP 回复时间
ip_address VARCHAR(45) IP地址
created_at TIMESTAMP 创建时间
site_configs（系统配置表）
字段名 类型 约束 说明
config_key VARCHAR(100) PK 配置键(主键)
config_value TEXT 配置值
value_type VARCHAR(20) 值类型
description VARCHAR(255) 配置描述
is_public BOOLEAN 是否公开
updated_at TIMESTAMP 更新时间
visit_logs（访问日志表）
字段名 类型 约束 说明
id BIGINT PK 主键，自增
path VARCHAR(500) NN 访问路径
referrer VARCHAR(500) 来源页面
ip_address VARCHAR(45) IP地址
user_agent VARCHAR(500) 用户代理
country VARCHAR(50) 访问国家
device_type VARCHAR(20) 设备类型
visited_at TIMESTAMP 访问时间
experiences（经历表）
字段名 类型 约束 说明
id BIGINT PK 主键，自增
type VARCHAR(20) NN 类型(work/education)
title VARCHAR(100) NN 职位/学位
company VARCHAR(100) NN 公司/学校
location VARCHAR(100) 地点
start_date DATE NN 开始日期
end_date DATE 结束日期
description TEXT 描述
achievements JSON 成就列表
display_order INT 显示顺序
is_active BOOLEAN 是否启用
files（文件资源表）
字段名 类型 约束 说明
id BIGINT PK 主键，自增
filename VARCHAR(255) NN 存储文件名
original_name VARCHAR(255) NN 原始文件名
mime_type VARCHAR(100) NN MIME类型
size BIGINT NN 文件大小(字节)
url VARCHAR(500) NN 访问URL
storage_provider VARCHAR(50) 存储提供商
storage_path VARCHAR(500) 存储路径
uploaded_by BIGINT FK 上传者ID→users
created_at TIMESTAMP 创建时间
integration_cache（API缓存表）
字段名 类型 约束 说明
cache_key VARCHAR(100) PK 缓存键(主键)
data JSON NN 缓存数据
last_updated TIMESTAMP NN 最后更新时间
expires_at TIMESTAMP NN 过期时间
ttl_seconds INT 生存时间(秒)
social_links（社交链接表）
字段名 类型 约束 说明
id BIGINT PK 主键，自增
platform VARCHAR(50) NN 平台名称
icon VARCHAR(50) 图标名称
label VARCHAR(50) NN 显示标签
url VARCHAR(255) NN 链接地址
username VARCHAR(100) 用户名
display_order INT 显示顺序
is_active BOOLEAN 是否启用

约束说明： PK 主键   FK 外键   UK 唯一   NN 非空
